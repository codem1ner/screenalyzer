The Window Object

The innerWidth, innerHeight, outerWidth, and outerHeight properties

The inner properties represent the viewport dimensions (in CSS pixels) and change with zoom level or browser window size. At 100% zoom, CSS pixel size = device pixel size.

The outer properties represent the browser window dimensions (in CSS pixels) and change with browser window size. They correspond to screen dimensions (resolution) when the browser window is maximised. They are unaffected by zoom level
